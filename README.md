# Sistema de Gestão de Mineradora

Sistema web desenvolvido para gerenciamento de equipamentos, cidades, funcionários e serviços.

## Tecnologias

- React
- Vite
- Node.js
- Express
- Cors
- Dotenv
- Supabase
- Netlify
- Render

## Deploy

Frontend:
https://sistemamineradoratrabalhoweb.netlify.app

Backend:
https://trabalho-web-di1f.onrender.com

## Como executar localmente

### Backend

cd Backend
npm install
npm start

### Frontend

cd Frontend
npm install
npm run dev
